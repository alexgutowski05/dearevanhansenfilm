How does it work?

  • I am using Adobe Premiere Pro CC 2019 and 2020 to edit footage.

  • I am using Adobe Audition CC 2019 and 2020 to record and edit audio

  • I am using Adobe Photoshop CC 2019 and 2020 to design characters

  • I am using Adobe Character Animator CC 2020 to make the people move

  • I am using Adobe Dreamweaver CC 2019 and 2020 to code this god-awful website.

  • I am also using Notepad + + to code.

  • I am using Audacity because it's free, and who doesn't like free stuff. (and because I actually know how to use it. I don't know how to use Audition.)

  • I am using OBS Studio to capture my screen.

  • I am using Photodex ProShow Producer and Photopia to make slideshow videos. (RIP ProShow 😖)

  • I am using Google Domains to purchace the domain www.dearevanhansenfilm.com

  • I am using GitHub to host my code online and be able to access it

  • I am using Netlify to assign the GitHub repo to the domain.
